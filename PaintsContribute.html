Contributing Paints

To contribute your own paint to the catalog, use the -submitDecal command in any discord server with Sprocket Bot.

This includes the Sprocket Official Discord.

Decals need to meet the following criteria:

- Files must be a .png
- Files should not exceed 1MB in size
- Decals need to be compliant with Discord TOS (no hateful symbols, NSFW, etc.)
