Update 16

Gear Calculator can now calculate a first gear based on your desired climbing angle!

Gear Calculator now also uses a proper calculation formula, finally supporting up to 1600 transmission gears! Don't try to go above that.

I might be able to add more customizability in the future, such as a custom R value.

Updated the credits page to include contributions made to the decal catalog and gear calculator.


Update 15

August 2nd, 2023

A proper logo and home page have now been added. No more sussy amogus for a logo...

...A bit ironic, isn't it?

Anyways: the Youtube Community Clash has now officially ended! Thanks to everyone who joined in on our contest!

To allow for other contests to make use of our tool, I have now converted the tank selector page to a standard selection page, but now with a "contest code." Contest hosts can select any code phrase or ID to use for their contest, and require participatns to apply the code when selecting their tank. This will permit variety between contests, while granting consistency and accuracy to the contest hosts themselves!



Update 14

August 1st, 2023

Gear Calculator has been added!

No more guessing what first and last gears you should use, this calculator does it all for you! Simply input a few stats about your tank, and Gear Calculator will give you an optimized transmission setup for your tank that allows you reach the highest speed possible!

Note: currently the calculator currently has issues with transmissions using more than 11 gears.

Update 9

July 8th, 2023

Finally outpaced Satisfactory in update numbers!

There has been an update to the rules for Toyotas and IFVs

Toyotas can now fall under the open-top classification, meaning you're able to ignore some space limitations. IFVs are now also allowed to be wheeled, meaning their tracks can be file edited to be invisible.

Remember that cheating is still not permitted. Things such as turret boosters, hiding compartments inside the vehicle, and non-permitted file editing isn't allowed.
