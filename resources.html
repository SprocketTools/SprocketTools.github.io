Fixing the broken Custom Battle Interface

Press win + R to open the run dialog. Paste the following, then hit enter:
%userprofile%\AppData\LocalLow\HD\Sprocket\
The custom battle configuration file is named CustomBattleSetup.json. Delete this file and restart the game, and your custom battle screen should be fixed.
